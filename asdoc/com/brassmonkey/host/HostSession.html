Packagecom.brassmonkey.host
Classpublic class HostSession
InheritanceHostSession flash.events.EventDispatcher

This class manages a server host session for micro-transactions.

Except for getting the user-icon, to use a host session, you must be a registered developer, and have server-side scripting capabilities.(jsp/php/etc)
