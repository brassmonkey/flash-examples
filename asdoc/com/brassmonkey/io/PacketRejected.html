Classpublic class PacketRejected
InheritancePacketRejected Error

Provides a means to halt packet processing and optionally kick the client that sent the packet.

By adding a listener to the BMApplication instance, you will recieve packet events of all types. If the handler of the packet data event received from the bmapplication throws a PacketRejected error, the packet will be dropped from further processing such as subsequent touch events or control button invokes. If the type is DISCONNECT_SENDER, the device will be removed from the host session.
