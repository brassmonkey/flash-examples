setInitialButtonState()method 
public function setInitialButtonState(obj:Object):void

For storing a local copy of a devices button state, pass an empty dynamic object to this method.

The object will be initialized with the names of the buttons as properties with value of 'up'. The BMApplicarion class is responsible for doing this and updating the button state. A full set of button state copies are kept by the BMApplicarion instance. In addition, DeviceEvent.SCHEME_BUTTON is dispatched by the individual devices when a listener is applied. The event.value will be the object with the local copies of the button states for that device.
