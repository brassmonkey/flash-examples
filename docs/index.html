ControlSchemes

Demonstrates two different topics related to Control Schemes. Hit Rects and Sample Modes.

Hit Rects

This example shows how to setup a button that has hit area that is separate from it's visual location.

To see the demo Right-click on the ButtonHitRect.as in Flash Builder and select Debug as Web Application.

See: Simple Hit Rect

Image Sample Modes

This example shows how to make an image that has is enlarged can either maintain a pixelated look or smoothed.

To see the demo Right-click on the SampleDemo.as in Flash Builder and select Debug as Web Application.

See: Image Sample Modes
