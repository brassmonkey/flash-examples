Examples


Boot Camp

Boot Camp explains some basic concepts about brass monkey. How it uses the flash eventing system and client-handlers. 20 minutes is the expected time for developers who are using an API for the first time. Just like working with FMS or any other asynchronous service for the first time, there are a few basic things to go over. The two tutorials are 1)a simple gravity controlled drawing application and 2)a monkey animation that dances with your device's position(its creepy). See: Brass Monkey In 20 Minutes

Control Schemes

Several different topics related to Control Schemes are covered. From advanced methods that change the Hit Rects of the buttons or alter the Sample Mode used while scaling to the different screen sizes, to the text components and aspect ratio API that help your design to be presented in the best possible way.

Hit Rects

This example shows how to setup a button that has a sensor-area larger than it's' visual size.

Image Sample Modes

Button and controls graphics may be scaled to fit different screen sizes. There are two ways to designate how they are interpolated. This demo sets up a dramatically scaled button and allows you to toggle the image resampling-mode by pressing it.

Try the demo


Aspect Ratio API

An in-line explanation of the aspect ratio API within a simple demo to help get your designs to look correctly on different screen sizes. It also explains how your design coordinates will define the finger-touch values received across all screen sizes.

Try the demo

Cookies

The Brass Monkey API includes three persistence methods for setting and reading session attributes. The first method is purely host-side, and can be written to and read from synchronously without delay. The second method is client-side and persists between sessions. The third method is server side and covered in the Host Sessions demo.

Leap Drop

A two player demo game. One player is a mad bomber, and the other player is a bomb-catching bucket. In addition to showing basic two-player logic, the javascript portion of the demo looks for a Leap-Motion hand-detection device, and alows the bomber's left-right movements to be controlled by hand gestures.

Trial Mode

Registered developers are able to offer premium content, and allow trial play or restrict assets.

The Host Session

Here we show how to get the user profile-icon. When your device connects to the host, it retrieves the icon you have picked. Registered developers are also able to create a secondary host session with their secret api key. The secondary session allows you to have micro-transactions in game and offer incentives (such as poker chips) that persist between games
